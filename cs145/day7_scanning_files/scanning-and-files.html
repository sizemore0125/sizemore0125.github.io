Scanning and Files

Last time, we created a Book class. We are assuming that this will be a part of a library management system that we are building. 

class Book {
	String title;
	int year;
	static int totalBooks = 0;

	Book(String title, int year) {
		this.title = title;
		this.year = year;
		totalBooks++;
	}

	void displayInfo() {
		System.out.println("Title: " + title);
		System.out.println("Publication Year: " + year);
	}

	static boolean compareTitle(Book book1, Book book2) {
		return book1.title.equals(book2.title);
	}
}

Perhaps we don't want to hard-code every book into our library management system, and instead we want to store all of our book information in a text file. In order to do this, we will need to know how to read from a file, so that we can instantiate an instance of the Book class based on the data in our file.

Let's assume our data is stored in text files with this format in a file called book_data.txt

title:Moby-Dick
year:1851
author:Herman Melville

title:Walden
year:1854
author:Henry David Thoreau

Reading From a File

To read information from files, we use the Scanner class. To read a file, you pass a File object to the Scanner constructor.

To create a File object, we need to give a file path to the File constructor:

String fileName = "book_data.txt";
File bookFile = new File(fileName);

*Note: The File class has many useful methods. Find them here.*

Now that we have our file object, we can use the Scanner class to read the file:

Scanner sc = new Scanner(bookFile);

All together so far:

import java.util.Scanner;
import java.io.File;
import java.io.IOException;

public class LibraryReader {
	public static void main(String[] args) throws IOException  {
		String fileName = "book_data.txt";
		File bookFile = new File(fileName);
		Scanner sc = new Scanner(bookFile);
		// Parsing the file code goes here.
	}
}

*Note: When using the Scanner class to read from files in Java, you must address potential exceptions that may occur. Specifically, reading a file with Scanner requires handling IOException*

Now, we can use this Scanner object to parse our file. To do this, we can use some common methods defined in the Scanner class.

Method Description
next() Reads and returns the next token (word) as a String.
nextLine() Reads and returns the entire line as a String.
hasNext() Returns true if there is another token to read.
hasNextLine() Returns true if there is another line to read.
close() Closes the Scanner, releasing any resources it holds.

Let's write some code that can parse our input file. We will begin by reading each line of our text file as long as there is a line to read. To do this, we has use the hasNextLine method in the Scanner class.

For example:

while (sc.hasNextLine()) {
	// Parse Lines
}

Now we can use the nextLine method to actually read the line from the file.

For example:

String line;
while (sc.hasNextLine()) {
	line = sc.nextLine();
	System.out.println(line);
}

Which will display:

title:Moby-Dick
year:1851
author:Herman Melville

title:Walden
year:1854
author:Henry David Thoreau

Now that we have read each line of our text file, we need to parse that file so that we can create an a Book object:

String title;
String year;
String author;
while (sc.hasNextLine()) {
	title = sc.nextLine();
	year = sc.nextLine();
	author = sc.nextLine();

	// Prevents an error on the last book in our file, 
	// as there will be no new line character. 
	if (sc.hasNextLine()) {
		sc.nextLine(); // Skips new line character after the book details.
	}

	System.out.println("%s %s %s%n%n", title, year, author);
}

Which will display:

title:Moby-Dick year:1851 author:Herman Melville

title:Walden year:1854 author:Henry David Thoreau

Now we have parsed the file and stored our book information in the title, year, and Author variables. However, we have a problem, as each of these strings has a informational prefix that we do not want. To remove these, we can use the replace method in the String class:

String title;
String year;
String author;
while (sc.hasNextLine()) {
	title = sc.nextLine().replace("title:", "");
	year = sc.nextLine().replace("year:", "");
	author = sc.nextLine().replace("author:", "");

	// Prevents an error on the last book in our file, 
	// as there will be no new line character. 
	if (sc.hasNextLine()) {
		sc.nextLine(); // Skips new line character after the book details.
	}

	System.out.println("%s %s %s%n%n", title, year, author);
}

Which will display:

Moby-Dick 1851 Herman Melville

Walden 1854 Henry David Thoreau

Now, we can use these values to instantiate a Book object:

String title;
String year;
String author;
while (sc.hasNextLine()) {
	title = sc.nextLine().replace("title:", "");
	year = sc.nextLine().replace("year:", "");
	author = sc.nextLine().replace("author:", "");

	// Prevents an error on the last book in our file, 
	// as there will be no new line character. 
	if (sc.hasNextLine()) {
		sc.nextLine(); // Skips new line character after the book details.
	}

	Book book = new Book(title, Integer.parseInt(year));
	book.displayInfo();
}
sc.close()

Which will display:

Title: Moby-Dick
Publication Year: 1851
Title: Walden
Publication Year: 1854

*Note: Our year variable was a String object, so we needed to convert it to an int as our Book object's constructor expects the year parameter to be of type int*
*Note: Once we are done with our Scanner object, it is good practice to close it using the close method, as we do not want it to continue taking up resources.*

This was an example of how you can read a file, and format the contents of that file so that it works for our library management system that we are developing. In the future, perhaps we would want to store each Book object in some data structure. But, for now, we will simply use our display method.

However, perhaps we also want to create an interface that allows a user to manually enter in some book information and have it create a book object. Let's do that now!

Reading input

To read input from a user, the Scanner class is also used. However, instead of passing a File object to the Scanner, you pass System.in.

Much like before, we must instantiate a Scanner object:

Scanner sc = new Scanner(System.in);

And now we can use various Scanner methods to read input from the user.

Method Description
next() Reads and returns the next token (word) as a String.
nextLine() Reads and returns the entire line as a String.
nextInt() Reads and returns the next token as an int.
nextDouble() Reads and returns the next token as a double.
nextFloat() Reads and returns the next token as a float.
nextLong() Reads and returns the next token as a long.
nextBoolean() Reads and returns the next token as a boolean.

Each of these methods will return a different type from the user. Let's use the nextLine method to grab the title and the year of publication of our book. 

Scanner sc = new Scanner(System.in);

System.out.print("What is the book title: ");
String title = sc.nextLine();

System.out.print("What is the year of publication: ");
int year = Integer.parseInt(sc.nextLine());

Book book = new Book(title, year);
book.displayInfo();

sc.close();

Here is example use of this program:

What is the book title: Moby-Dick
What is the year of publication: 1851
Title: Moby-Dick
Publication Year: 1851

*Note: I do not use the parseInt method because it does not consume a new line character, and it causes issues when you are trying to read multiple inputs from a user. Instead, I only use readLine and I deal with parsing myself.*

We now have a program that prompts a user for an input and instantiates a Book object based on what the user enters. Perhaps now we want our program to also write to a file based on what the user enters, so that they can use that file as a method of tracking books in the library.

Writing Files

Let's begin by adding a new field to our book class author and adding a method that returns a String representation of our book that we will write to a file. 

class Book {
	String title;
	int year;
	String author;
	static int totalBooks = 0;

	Book(String title, int year, String author) {
		this.title = title;
		this.year = year;
		this.author = author;
		totalBooks++;
	}

	void displayInfo() {
		System.out.println("Title: " + title);
		System.out.println("Publication Year: " + year);
	}

	String getBookString() {
		String bookString = "title:" + title + 
			"\nyear:" + year + 
			"\nauthor:" + author + "\n";
		return bookString
	}

	static boolean compareTitle(Book book1, Book book2) {
		return book1.title.equals(book2.title);
	}
}

When I execute this code:

Book book = new Book("Moby-Dick", 1851, "Herman Melville");
System.out.println(book.getBookString());

It will display:

title:Moby-Dick
year:1851
author:Herman Melville

Now we have a method to simply get a string representation for out book that we would want to write to a file.

In Java, the FileWriter class is commonly used to write text to a file.

For example:

import java.io.FileWriter;
import java.io.IOException;

public class WriteBook {
    public static void main(String[] args) throws IOException {

		Book book = new Book("Moby-Dick", 1851, "Herman Melville");

	    FileWriter writer = new FileWriter("book_storage.txt");

		writer.write(book.getBookString() + "\n");
		
        writer.close();
    }
}

This code will write this text to book_storage.txt:

title:Moby-Dick
year:1851
author:Herman Melville

This looks good so far, but lets extend our interface so that it will write each book a user enters to a file:

import java.io.FileWriter;
import java.io.IOException;
import java.util.Scanner;

public class WriteBook {
    public static void main(String[] args) throws IOException {
		Scanner sc = new Scanner(System.in);
	    FileWriter writer = new FileWriter("book_storage.txt", true);
	    
	    String title;
	    int year;
	    String author;
		while (true) {
			System.out.print("What is the book title: ");
			title = sc.nextLine();
			
			// Quit when the user enters "q"
			if (title.equals("q")) {
				break;
			}
			
			System.out.print("What is the year of publication: ");
			int year = Integer.parseInt(sc.nextLine());
			
			System.out.print("What is the author: ");
			author = sc.nextLine();
			
			Book book = new Book(title, year, author);
			writer.write(book.getBookString() + "\n");
		}
		
		sc.close();
        writer.close();
    }
}

Now, we have a program that prompts a user to enter in book information, it creates a Book object and uses the getBookString to format the book object's information into a meaningful string, then it will append that to a file. Later, we can use our original file parsing program to read that file and instantiate the same Book object again!

*Note: To add content to an existing file without overwriting it, use the FileWriter constructor with the append parameter set to true.
(e.g., FileWriter writer = new FileWriter("book_storage.txt", true);)*
